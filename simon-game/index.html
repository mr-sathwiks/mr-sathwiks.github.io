This game is based on a 90's classic game Simon
which is structured on a memory game palyed by kids.
Instruction's to play:

  1. There are 4 buttons, with 4 diffrent colors.
  2. At level 1 a button will be highlighted. You have to press button being highlighted and also memorize it.

  3. For the nexet level a new button will be highlighted. You have to first press the button highlighted from the previous level and then the current button.

    e.g:
    the button highlighted for level 1: RED,
    the button highlighted for level 2: YELLOW,
    the button highlighted for level 3: BLUE,
    the button highlighted for level 4: RED,

    if the button highlighted for level 5 is GREEN,
    you have to first press the button in the previous order: RED, YELLOW, BLUE, RED and then GREEN.

  4. Make sure to press the button in the order itself
  5. If you fail to press the button in the shown pattern, game will be terminated.
  6. The pattern will keepchanging for each trail.
  7. The pattern is being generated randomly, they cannot be predicted.
